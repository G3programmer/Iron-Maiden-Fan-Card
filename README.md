## 💀 Sobre o Projeto

Este projeto é uma aplicação web desenvolvida em **Vue 3 (Vite)** que vai além de uma simples wiki informativa. A proposta é criar uma jornada imersiva para o usuário: ao navegar pelos cards dos álbuns clássicos do Iron Maiden, uma **atmosfera única** que é ativada através de narrações dedicadas a personalidade da banda, exibição das versões do mascote **Eddie** e trilha sonora imersiva de cada época da banda.

### ⚡ Recursos Principais:
* **Grid Dinâmico:** Renderização inteligente de álbuns utilizando as diretivas do Vue 3 (`v-for`).
* **Navegação Imersiva:** Layout dark customizado inspirado na estética clássica e cores icônicas do Iron Maiden.
* **Experiência de Áudio Sequencial:** Narração bem trabalhada para introduzir a lore de cada álbum antes de apresentar as faixas musicais.
* **Otimização de Assets:** Gerenciamento eficiente de imagens e álbuns estáticos via Vite.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas e tecnologias modernas de desenvolvimento web:

* **[Vue.js 3](https://vuejs.org/):** Framework JavaScript progressivo utilizando a reatividade da `Composition API` (`<script setup>`).
* **[Vite](https://vitejs.dev/):** Ferramenta de build ultra-rápida para o ecossistema frontend.
* **JavaScript (ES6+):** Lógica assíncrona para controle e sequenciamento de áudio.
* **CSS3 Customizado:** Estilização responsiva com variáveis de cores (neon verdes e roxos sobre fundo dark).
